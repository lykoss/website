The following is a list of commands in Werewolf.


Before a game

Command Function
!ping Pings everyone to get their attention. This list excludes users who are away. (please don't manually ping users)
!wait Delays the start of a game with 20 seconds. Can only be used a few times per game.
!join Join a game.

During a game

Command Function
!roles Lists all roles. Roles playing in the current game are prefixed by a bold character.
!stats Lists alive players and roles as well as the game status (day or night).
!lynch Votes to lynch a player

Role commands

Command Role Day/night Function
/msg lykos kill target Wolf, werecrow Night Votes to kill target.
/msg lykos see target Seer Night Discover the role of target (can only be used once every night).
/msg lykos visit target Harlot Night Visits target at night.
/msg lykos observe target Werecrow Night Observes target. At the end of the night, the werecrow is informed whether target was sleeping or not.
/msg lykos id target Detective Day Investigates the role of target and returns results immediately.
/msg lykos guard target Guardian angel Night Guards a player from the wolves.

Wolfgame Commands

Role-Specific Commands

/MSG lykos kill
Kill someone (wolf).
/MSG lykos see
Get the role of someone at night (seer).
/MSG lykos id
Get the role of someone during day (detective).
/MSG lykos visit
Visit someone at night (harlot).
/MSG lykos guard
Guard someone (Guardian Angel).
!shoot
Shoot someone (Gunner).
