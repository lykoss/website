The following is a list of commands in Werewolf.


General commands

Command Function
!away Marks you as away so the bot doesn't ping you. Stores your cloak, not nick. Can be done in the channel or private message.
!back Removes you from the away list.
!simple Allows for messages to be recieved as notice. If you have already set this, then setting it again will revert messages to privmsgs.
/msg lykos !simple See !simple

Before a game

Command Function
!ping Pings everyone to get their attention. This list excludes users who have set !away or /away. (please don't manually ping users)
!wait Delays the start of a game with 20 seconds. Can only be used 3 times per game.
!join Join the game.
!quit or !leave Leave the game.
!start Starts the game

During a game

Command Function
!roles Lists all roles. Roles playing in the current game are prefixed by a bold character.
!myrole Reminds you what role you are, in case you forget.
!stats Lists alive players and roles as well as the game status (day or night).
!lynch player or !vote player Votes to lynch a player

Role commands

Command Role Day/night Function
/msg lykos kill target Wolf, werecrow Night Votes to kill target.
/msg lykos see target Seer Night Discover the role of target (can only be used once every night).
/msg lykos visit target Harlot Night Visits target at night.
/msg lykos observe target Werecrow Night Observes target. At the end of the night, the werecrow is informed whether target was sleeping or not.
/msg lykos id target Detective Day Investigates the role of target and returns results immediately.
/msg lykos guard target Guardian angel Night Guards a player from the wolves.
!shoot target Gunner Day Shoots target.
